"ChatWithDave" AI Assistant

A personalized and secure AI assistant built with Python, Flask, and Gemini AI, demonstrating a full-cycle development process from design to production deployment.

The Three-Month Journey

This project was a three-month marathon, taking an idea from a design concept to a fully deployed, production-ready application. It showcases my end-to-end skills in design, frontend and backend development, and cloud infrastructure management.

Week 1: Design & Prototyping

The journey began in Figma, where I spent a week designing a clean, intuitive, and engaging user interface. I focused on creating seamless user flows for registration, login, and the core chat functionality to ensure a great user experience from the start.

Figma

Weeks 2-4: Frontend Development

With the Figma designs as a blueprint, I spent three weeks building the client-side of the application. I translated the static designs into a fully responsive and interactive web interface using semantic HTML, modern CSS, and vanilla JavaScript to handle real-time UI updates.

HTML5 CSS3 JavaScript

Month 2: Backend & AI Integration

The next month was dedicated to building the application's core logic. I developed a robust backend using Python and Flask, integrating the Gemini AI API for conversational intelligence. This phase included setting up the database, implementing user authentication, and creating the API endpoints to connect the frontend and backend.

Python Flask Gemini AI SQLAlchemy

Month 3: Testing, Iteration & Deployment

The final month was focused on refinement and deployment. I rigorously tested the application, fixing bugs and improving the user experience based on UX principles. This included perfecting the login, sign-up, and message history flows. Finally, I deployed the application to a production environment.

Render Supabase Gunicorn

Key Features

Interactive Chat Interface

A clean, responsive, and modern web interface for conversing with the AI, designed for a great user experience on any device.

Dual Authentication System

Supports both secure local accounts (email/password) and seamless sign-in with Google OAuth 2.0 for user convenience.

End-to-End Encrypted History

For password-based accounts, chat history is encrypted in the database using a key derived from the user's password, ensuring privacy.

Profile & Password Management

Users can update their display name and manage their account security with a "forgot password" reset functionality.

Gemini AI Integration

Utilizes Google's powerful and efficient `gemini-1.5-flash-latest` model for natural and context-aware conversations.

Guest Mode

Allows new users to try the chatbot with a message limit before registering, providing a frictionless trial experience.

Architecture & Tech Stack

The application is built on a modern, scalable stack and deployed on a PaaS for easy management. Initially deployed on AWS, the project was migrated to Render and Supabase to optimize for a personal portfolio use case, demonstrating adaptability across different cloud environments.

Backend

  • Language: Python
  • Framework: Flask
  • AI Model: Gemini 1.5 Flash
  • Database: PostgreSQL (Supabase)
  • ORM: SQLAlchemy

Frontend

  • Markup: HTML5
  • Styling: CSS3
  • Interactivity: JavaScript
  • Design: Figma

Deployment

  • Platform: Render (PaaS)
  • WSGI Server: Gunicorn
  • Database Provider: Supabase
  • Version Control: Git & GitHub
