Automated Weather & Energy Analysis Pipeline

An interactive data application designed to improve energy demand forecasting. By combining weather data with consumption patterns, this solution helps utilities optimize power generation, reduce operational costs, and enhance grid stability, turning complex data into actionable business intelligence.

From Configuration to Insight: An Interactive Workflow

This project turns a complex data pipeline into a simple, three-step interactive process, all handled within the dashboard UI.

1. Dynamic Configuration in the UI

Instead of command-line scripts, all controls are built into the interface. Users can find reliable weather stations for new cities, see real-time data validation warnings (e.g., for states with insufficient data), and save new configurations directly within the dashboard's YAML editor.

Streamlit YAML API

2. One-Click Pipeline Execution

The 'Refresh All Data' button is the magic trigger. It executes a master script (`run.py`) that orchestrates the entire backend process: fetching raw data from NOAA and EIA APIs, and then processing, cleaning, and validating it for analysis.

Python run.py Data Fetching

3. Comprehensive Analysis Dashboard

Once the pipeline completes, the dashboard instantly updates with fresh insights across multiple views: a geographic map of hotspots, time-series comparisons, correlation plots, usage heatmaps, and a dedicated data quality report to ensure full confidence in the results.

Streamlit Plotly Pandas

Analysis Highlights

Dynamic Geographic Hotspots

Instantly map new energy demand hotspots across all configured cities after a data refresh.

Time Series Analysis

Compare consumption trends across all locations and identify weekly patterns with weekend highlighting.

Correlation Analysis

Get the hard numbers. Quantify the strong link between temperature and energy use with regression analysis.

Usage Patterns Heatmap

Reveal crucial daily and weekly consumption patterns based on temperature ranges for any selected city.

Self-Contained Application

No command line needed. Configure, refresh, and analyze data entirely within the interactive UI.

Built-in Data Validation

A dedicated report and real-time warnings ensure that all analysis is based on clean, reliable, and sufficient data.
